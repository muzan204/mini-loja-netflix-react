##⚙️ Como rodar o projeto
```bash
# Clonar o repositório
git clone https://github.com/SEU_USUARIO/mini-loja-netflix-react.git

# Entrar na pasta
cd mini-loja-netflix-react

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```
